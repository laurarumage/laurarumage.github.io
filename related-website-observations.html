I chose a similar web page from The Bark, a dog magazine. The page is an article listing things the author loves about her dog.

Website Comparison

This web site's page has both qualities that are good and qualities that could be improved:

  • Good:
    • The author is talking about dogs, and dogs are wonderful.
    • The ordered list on the page is set up properly with a list tag, specifically <ol>.
    • Each item on the ordered list mentioned above is created as a distinct list item using the semantic tag <li>.
  • Room for improvement:
    • I think the page's main navigation should have been contained in a semantic <nav> tag. The navigation is actually setup as a table.
    • In researching where the navigation table mentioned above begins, I also determined that the entire visible web page is contained within a table! It is interesting to still see a website with table layout while table-less design is widely favored these days. Tables were never intended as a layout tool--tables were intended for the display of tabular data!
    • The article does not use header tags to semantically define header content. What should be created as the <h1> tag has been created as a div with a class assignment to provide meaning.

Overall, there are some areas that this web site's page could improve, namely in the area of including more semantic tags to provide additional meaning to the content.
