Sir Digby Chicken Caesar

Basic Info

Digby is an Australian Cattle Dog mix rescue dog with the most magnificent underbite this world has ever seen. He lives in Chicago, IL with his mom and dad, who are both humans.

Did you know?

"An underbite is the projection of the lower teeth beyond the upper. "
- Oxford Dictionaries

In the case of Digby's underbite, his teeth do not even fit inside his face.

Interests

Digby's interests include:

  • Dissecting stuffed animals
  • Sighing dramatically when bored
  • Watching pizza cook in the oven
  • Avoiding dog furniture

In The Media

Digby is famous in a Milwaukee, Wisconsin, suburb, where his stunning smile graces an online community newspaper article. It is but a coincidence that his uncle Jeff, a human person, is the article's author.

Related website observations

I chose a similar web page from The Bark, a dog magazine. The page is an article listing things the author loves about her dog.

This page has both good and bad qualities:

  • Good:
    1. The author is talking about dogs, and dogs are wonderful.
    2. The ordered list on the page is set up properly with a list tag, specifically <ol>.
    3. Each item on the ordered list mentioned above is created as a distinct list item using the semantic tag <li>.
  • Bad:
    1. I think the page's main navigation should have been contained in a semantic <nav> tag. The navigation is actually setup as a table.
    2. In researching where the navigation table mentioned above begins, I also determined that the entire visible web page is contained within a table! It is interesting to still see a website with table layout while table-less design is widely favored these days. Tables were never intended as a layout tool--tables were intended for the display of tabular data!
    3. The article does not use header tags to semantically define header content. What should be created as the <h1> tag has been created as a div with a class assignment to provide meaning.

Overall, there are some areas that this web page could improve, namely in the area of including more semantic tags to provide additional meaning to the content.
